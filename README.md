# Python Practice — 3 Month Learning Tracker

I use this repository as a progress tracker for learning Python from scratch to a project-building level within 3 months. After completing each topic, mark it as done by changing `[ ]` to `[x]`.

---

## 📅 Month 1: Foundations

### Week 1 — Syntax & Basics
- [x] Variables, data types (int, float, str, bool)
- [x] print(), input(), comments
- [x] Operators (arithmetic, comparison, logical)
- [ ] Project: Simple calculator + temperature converter

### Week 2 — Control Flow
- [ ] if / elif / else
- [ ] for loops
- [ ] while loops
- [ ] break, continue
- [ ] Project: Number guessing game + FizzBuzz

### Week 3 — Data Structures
- [ ] Lists, tuples, sets, dictionaries
- [ ] Indexing & slicing
- [ ] List comprehensions (basic)
- [ ] Project: To-do list app (console) + word counter

### Week 4 — Functions
- [ ] def, parameters, return values
- [ ] Default args, *args, **kwargs
- [ ] Scope (local vs global)
- [ ] Project: Library / inventory system

---

## 📅 Month 2: Intermediate

### Week 5 — Error Handling & File I/O
- [ ] try / except / finally
- [ ] Reading/writing files (txt, csv)
- [ ] Project: Log file analyzer

### Week 6 — Object-Oriented Programming
- [ ] Classes, objects, `__init__`
- [ ] Inheritance, methods
- [ ] Project: Bank account / student class system

### Week 7 — Modules & Libraries
- [ ] import, pip install
- [ ] Standard library (os, sys, datetime, random)
- [ ] Virtual environments (venv)
- [ ] Project: Small CLI tool using an external library

### Week 8 — Working with Data
- [ ] JSON handling
- [ ] requests (basic API calls)
- [ ] Intro to pandas (optional)
- [ ] Project: Weather app using a free API

---

## 📅 Month 3: Apply & Specialize

### Week 9-10 — Pick a Direction
- [ ] Choose: Web dev (Flask/Django) / Data & Automation (pandas, scraping) / Scripting
- [ ] Follow focused mini-course in chosen direction
- [ ] Build 2 small exercises

### Week 11 — Real Project
- [ ] Plan project scope
- [ ] Build core features
- [ ] Add error handling
- [ ] Push to GitHub

### Week 12 — Polish & Review
- [ ] Refactor & clean up project
- [ ] Read 2-3 open-source repos on GitHub
- [ ] Solve 5+ beginner problems (LeetCode/HackerRank easy)
- [ ] Final portfolio project ready

---

## 🧠 Fast-Learning Reminders
- Type code by hand — never copy-paste
- Build more than you watch (60-70% hands-on)
- Finish at least 1 small project every week
- Read every error message carefully
- Commit to GitHub after each topic/practice file
